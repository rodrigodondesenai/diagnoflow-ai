# Agent Prompts

## Prompt de trabalho para esta fase

O projeto foi estruturado para separar claramente:

- captura de sintomas;
- geração de diagnóstico mockado;
- renderização de relatório técnico;
- documentação da futura camada de IA.

## Diretrizes adotadas

- Não integrar LLM, API externa ou inferência real nesta fase.
- Manter a mesma jornada de uso que será reaproveitada na segunda entrega.
- Preparar contratos de dados estáveis entre frontend e backend.
- Destacar explicitamente na interface e na API que o diagnóstico é simulado.

## Futuro prompt do agente

O rascunho do futuro system prompt foi mantido em `prompts/future-system-prompt.txt`, sem uso em tempo de execução nesta entrega.
